# Front-end Page Project

## Tecnologias utilizadas

### Pré-processadores

* SASS

### Task Runner

* Gulp

### Gerenciador de Pacotes

* Node.js
* Yarn

## Rodando o projeto

```sh
# Clone o repositório
git clone https://github.com/Hysumi/front-end-BEM.git

# Instale as dependencias.
npm install

# Rodar localmente.
yarn serve
```
